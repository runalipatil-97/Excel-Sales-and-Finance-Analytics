# Excel-Sales-and-Finance-Analytics

Overview

This project focuses on analyzing sales and finance data to provide actionable insights for business decision-making. The goal is to consolidate raw sales data, generate meaningful performance reports, and identify trends across products, customers, and regions.

Tools Used

MS Excel – Pivot Tables, Charts, Formulas

Power Query – Data cleaning, transformation, consolidation

Power BI (optional) – Dashboarding and visualization

Key Tasks Performed

Consolidated sales and financial data from multiple sources using Power Query.

Performed calculations for Net Sales, Cost of Goods Sold (COGS), Gross Margin, Growth %, and Profitability.

Created Pivot Tables to summarize data by customer, product, and region.

Built structured reports highlighting performance trends and key metrics.

Key Insights & Business Impact

Identified top-performing regions and products.

Highlighted areas where revenue growth was strong but margins were low, indicating potential cost issues.

Generated actionable reports to assist management in tracking performance and making data-driven decisions.

Project Outcome

This project demonstrates how structured data analysis can provide insights for sales strategy, financial planning, and operational efficiency. The reports created can directly support decision-making for business growth and profitability.
