# Excel-Sales-and-Finance-Analytics

Overview

This project focuses on analyzing sales and finance data to generate actionable insights and help management make data-driven decisions. The goal was to consolidate raw data, create structured reports, and identify performance trends across products, customers, and regions.

Tools & Technologies

MS Excel – Pivot Tables, Charts, Formulas

Power Query – Data cleaning, transformation, and consolidation

Power BI (optional) – Dashboarding and visualization

Key Tasks Performed

Consolidated sales and finance data from multiple sources using Power Query.

Calculated key metrics such as:

Net Sales

Cost of Goods Sold (COGS)

Gross Margin

Growth Percentage

Profitability by product and region

Created Pivot Tables and charts to summarize sales and financial performance.

Built structured reports highlighting trends and key insights.

Key Insights & Business Impact

Identified top-performing regions and products.

Highlighted areas where revenue growth was strong but profit margins were low, indicating potential cost or pricing issues.

Generated reports that allow management to quickly track performance and take informed business decisions.

Demonstrated how structured data analysis improves operational efficiency and supports strategic planning.

Outcome

This project shows how Excel and Power Query can be used to transform raw business data into actionable insights, enabling better decision-making in sales and finance operations.
Project Outcome

This project demonstrates how structured data analysis can provide insights for sales strategy, financial planning, and operational efficiency. The reports created can directly support decision-making for business growth and profitability.
